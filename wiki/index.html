Requirements

  • Java 17 or higher

Quick Start Guide

Get started with NotEnoughTinkering plugins in just a few minutes.

NotEnoughMatter Quick Start

Creating Your First Item

Follow these steps to create your first custom textured item:

  1. Go to the NotEnoughMatter Generator
  2. Upload a 16√ó16 PNG texture
  3. Fill in the item details (ID, Name, Material)
  4. Copy the HEX lines or download the YAML file

Using In-Game Commands

/nem create <id> <category>
/nem give <item_id> <player> <amount>
/nem list
/nem reload

Example

/nem create custom_pickaxe pickaxe_head
# Follow chat prompts to input texture lines
/nem give custom_pickaxe k0dev 1

NotEnoughRelics Quick Start

Creating Your First Relic

  1. Go to the NotEnoughRelics Generator
  2. Set the relic name, material, and rarity
  3. Add custom abilities and stats
  4. Upload a 16√ó16 texture (optional)
  5. Download the YAML configuration

Using In-Game Commands

/ner give <relic_id> <player> [amount]
/ner list
/ner info <relic_id>
/ner reload

Example YAML

id: ancient_sword
name: "Ancient Sword of Power"
material: DIAMOND_SWORD
rarity: LEGENDARY
lore:
  - "&7An ancient weapon forged in..."
  - "&7the fires of the Nether itself."
abilities:
  ability_1:
    trigger: RIGHT_CLICK
    effect: LIGHTNING
    cooldown: 10
    power: 5.0
